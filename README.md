# SariJay Traffic Automation

**AI-Powered Traffic Generation Platform**

> "Set Your Link. Let AI Bring The Traffic."

An intelligent traffic generation platform designed to help affiliate marketers, online business owners, and digital entrepreneurs drive large amounts of targeted website traffic automatically.

## Project Overview

SariJay automates traffic generation through AI, allowing users to:
1. Sign in
2. Paste their affiliate link
3. Enter keywords and niche
4. Activate automation

The AI handles everything else automatically.

## Getting Started

### Prerequisites
- Node.js 18+
- Docker & Docker Compose
- PostgreSQL 14+

### Installation

```bash
git clone https://github.com/ipcaffiliate1-maker/SariJay-Automation.git
cd SariJay-Automation
npm run install:all
docker-compose up
```
